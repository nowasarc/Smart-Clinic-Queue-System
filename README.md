# Smart Clinic Queue System 🏥

A Java-based Smart Queue and Appointment Management System designed for small clinics to improve patient flow, prioritization, and service efficiency.

---

## 📌 Features

- 🧾 Add patient information (ID, Name, Service)
- ⚡ Priority Queue for urgent patients
- 📋 Regular Queue (FIFO)
- 🔍 Search patient using Binary Search Tree
- 🕘 View patient history (Linked List)
- 🖥️ Simple GUI using Java Swing

---

## 🧠 Data Structures Used

- **Queue** – for regular patient flow (FIFO)
- **Priority Queue** – for urgent patients
- **Linked List** – for storing served patients (history)
- **Binary Search Tree (BST)** – for fast searching

---

## ⚙️ Technologies Used

- Java
- Java Swing (GUI)
- IntelliJ IDEA
